# Anix Landing

Статический продающий лендинг Anix без сборщика и внешних JavaScript-зависимостей.

## Локальный запуск

```powershell
python -m http.server 8080 --directory anix_landing
```

После запуска откройте `http://localhost:8080`.

## Структура

- `index.html` — контент и разметка
- `styles.css` — адаптивный дизайн и анимации
- `script.js` — появление секций, карусель экранов и микроинтеракции
- `assets/` — логотип и продуктовые скриншоты
