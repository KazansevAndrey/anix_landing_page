# Anix

Персональная ИИ-память для заметок, смыслового поиска, инсайтов и глубокого анализа.

## Лендинг

### [Открыть Anix на GitHub Pages →](https://kazansevandrey.github.io/anix_landing_page/)

Страница автоматически обновляется после каждого коммита в ветку `main`.

## Локальный запуск

```powershell
python -m http.server 8080
```

После запуска откройте `http://localhost:8080`.
